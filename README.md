# Gölge Kalkan Tanıtım Sayfası

Bu proje Hafta 1 HTML ödevi için hazırlanmış basit bir web sayfasıdır.

## İçerik

- `index.html`: Ana HTML sayfası
- `style.css`: Sayfanın görünüm dosyası
- `assets/golge-kalkan.svg`: Kahraman görseli

## Ödevde İstenenler

- Başlık kullanımı var: `h1`, `h2`, `h3`
- Paragraf kullanımı var: `p`
- Liste kullanımı var: `ul` ve `ol`
- Görsel kullanımı var: `img`
- Bağlantı kullanımı var: `a href`
- Form alanları var: `input`, `textarea`, `button`
- Formda `required` ve `placeholder` kullanıldı

## GitHub Pages Yayınlama

1. GitHub'da yeni bir repository oluştur.
2. Bu klasördeki dosyaları repository içine yükle.
3. Repository sayfasında `Settings` bölümüne gir.
4. `Pages` bölümünde branch olarak `main` seç.
5. Kaydettikten sonra GitHub Pages linkini eğitim grubuna gönder.
